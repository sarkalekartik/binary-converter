# binary-converter
A simple tool to convert decimal numbers to binary and back



Multi-Converter Tool
A lightweight utility for converting Binary, IPv4, and IPv6 addresses.
🚀 Features
Binary Converter: Seamlessly switch between decimal and binary formats.
IPv4 Tool: Handle 32-bit address conversions and subnetting basics.
IPv6 Tool: Manage 128-bit hex strings and shorthand notation.
📦 Installation & Usage
This tool is distributed as a portable .zip archive. No complex installation is required.
Download: Get the latest converter-tool.zip from the Releases section.
Extract: Right-click the zip file and select Extract All... (or use unzip in terminal).
Run:
Open the extracted folder.
Run the executable file (e.g., converter.exe or main.py).
